# SpiceShelf Recipe App – Frontend Only

## Project Overview
SpiceShelf is a responsive recipe web app built using HTML, CSS, JavaScript, and Bootstrap.

## Setup Instructions
1. Download the project.
2. Open `index.html` in a browser.

## Features
- Search recipes
- Filter by category
- View recipe details
- Responsive design

## Note
This is a frontend-only version (no PHP or database).
