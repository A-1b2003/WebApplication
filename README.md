# SpiceShelf Recipe App – Frontend Only

## Project Overview
SpiceShelf is a responsive recipe web application developed using HTML, CSS, JavaScript, and Bootstrap.  
It allows users to search, filter, and view recipes interactively.

## Setup Instructions
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.

## Features
- Search recipes by name
- Filter recipes by category
- View recipe details in a popup modal
- Responsive design for different screen sizes
- Image carousel for featured recipes

## Folder Structure
- index.html
- features.html
- contact.html
- css/
- js/
- images/

## Note
This is a frontend-only implementation. No backend or database is included.
